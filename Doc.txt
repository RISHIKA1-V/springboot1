Generating ssh key
1)open git bash anywhere in system.
2)ssh-keygen -t ed25519 -C "your_email@example.com"
  note:please just give enter for everything till u 
  get the key.

3)check ssh agent
   eval "$(ssh-agent -s)"
4)add identity
   ssh-add ~/.ssh/id_ed25519
5)copy ssh key 
    cat < ~/.ssh/id_ed25519.pub
